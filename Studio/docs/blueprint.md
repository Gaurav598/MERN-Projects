# **App Name**: TaskFlow

## Core Features:

- Task Management: Add, edit, and delete tasks with priority levels (Low/Medium/High), due dates, and category/tag system.
- View Tasks: View tasks by Today, Upcoming, All Tasks, Completed, By Category, and a basic Calendar view.
- Drag and Drop: Drag and drop reordering of tasks within each list.
- Task Completion: Mark tasks as complete with a smooth animation and haptic feedback. Undo delete (snackbar).
- Smart Filters: Apply smart filters to refine task lists based on priority, due date, category, etc.
- Search and Sort: Instant search for tasks and various sort options.
- Category Based Suggestions: Use the LLM to suggest useful tags/categories based on the entered text in the task description as a tool to allow faster organization.

## Style Guidelines:

- Primary color: Indigo (#4B0082), for a modern and calm feel.
- Accent color: Soft Purple (#A020F0) to provide visual interest and highlight key interactive elements.
- Background color: Off-white (#F5F5F5) for light mode, Charcoal (#333333) for dark mode, ensuring a comfortable reading experience.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to provide a clean, neutral look.
- Note: currently only Google Fonts are supported.
- Use a set of consistent, minimalistic icons for task categories and priority levels.
- Subtle animations for task completion, swipe actions, and screen transitions.
